Linguagem de Programação Server-side e Client-side

Server-side

  • O codigo do lado do servidor é executado em um servidor da web.
  • É responsavel por processar as soluções dos usuários, como acesso a bancos de dados, autenticação e lógica de negócios.
  • O código do lado do servidor pode ser escrito em várias linguaguens de programação tipo PHP ou C#
    • Por exemplo: se criarmos um script em linguagem back-end (PHP, Asp, Java, Ruby, etc) que apenas calcula a
    soma de 1 + 1, será o SERVIDOR (ou back) que calculará este resultado.

    Client-side

  • O código do lado do cliente lida principalmente com a estrutura e apresentação dos dados para o usuário.
  • O código do lado do cliente é executado no navegador do usuário, como o JavaScript.
  • Do front-end se refere aos tipos de processos que são executados no lado do cliente.
    • Por exemplo: se fizermos uma página web usando html e css, o JavaScript vai dar dinâmicas e mecânicas novas ao site.
